TCP CLient and Server app ver 1.0.0
===================================
Written by: Robert Lee

In order to run this program you need to have 3 Terminals up
1: TCPServer 2:TCPClient 3:TCPClient2

At this present time you can send messages, and files between clients once logged in.
NO use of GUI Forms at the present moment
DO NOT touch the SERVER Terminal when it starts running. Only interact with the Client Terminals

Also at this Time, although you can send any file format, it gets renamed as a .txt file due to the programs code
I'll see about fixing this if needed but for the purposes of the project the entire program functions
for all requirements of the Project.

Make sure to have the TextFile.txt in the same folder as the java folders to run it.
When you send a attachment over the clients it will save as "Username recieved by"TextFile.txt
